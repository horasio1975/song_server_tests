## requirements
1. python 3.6
2. working directory should look like:  
   projects    
       -songs_server  
       -songs_server_tests

## run
1. git clone https://github.com/horasio1975/song_server_tests.git
2. cd song_server_tests
3. pip install pipenv
4. pipenv install
5. pipenv shell
6. run tests: python -m pytest
